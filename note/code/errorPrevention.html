# 怎么看待犯错误这件事情

# 三个普遍看法

1 好的程序员不会写坏的代码，要不然他就是不够优秀

本身不认同，难免犯错

2 同样的错误不能犯第二次

是这么要求自己的。但是对于他人来说，应该更加宽容。对于团队来说，应该思考如何提供一种机制，以减少此类错误的发生。

3 一个人犯了错误不可怕，可怕的是不承认错误。

确实深入人心，从小老师、家长就是这么教育过来的。但是希望在犯错这件事情上，多一点对自己的谅解，以及对他人的宽容。因为一旦陷入自责中，对内损耗过于严重，而且容易自卑。而一旦陷入指责中，很可能一两个小时的时间就过去了，而那些有建设性的事情，我们可能没有意识或者时间去做。
