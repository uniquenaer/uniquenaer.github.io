# 简介

lin-docfork 自vuepress，拓展了一些功能使其适合林间有风团队风格。

# 快速上手

git clone https://git.coding.net/indexer/Lin-Doc.git // 拷贝代码

cd Lin-Doc    // 进入项目根目录

yarn          // 安装依赖

yarn dev      // 启动服务

# 编写规范

  • 页面展示在左侧 sidebar 的 title 书写格式，是在页面顶部写上：
---
title: 书写规范
---

  • # 第一个标题应该是一级标题，理想情况下应与文件名相同或几乎相同。

  • ## 其余标题应从 2 级开始

  • 标题后需添加一个空格

// bad
##标题

// good
## 标题
  • 标题和内容间需有一个空行
// bad
## 标题
本段内容
## 标题

// good
## 标题

本段内容

## 标题
  • 蓝色字体 在书写的时候，两侧需加上空格
// bad
其他文本`蓝色字体`其他文本

// good
其他文本 `蓝色字体` 其他文本

  • 代码段的需使用 Fenced code blocks 风格，如下所示：

    ```
console.log("");
```

  • 有序列表

    1. 第一条信息

    2. 第二条信息

    3. 第三条信息
