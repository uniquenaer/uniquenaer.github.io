# 用户接口

路由: user/register post 方法 参数:

name 说明 类型 作用
nickname 用户名 string ****
password 密码 string ****
confirm_password 确认密码 string ****
email 邮箱，可选 string ****
group_id 分组 id int ****

备注： 用户注册，管理员可用

发送：

{
  "nickname": "pedro",
  "password": "123456",
  "confirm_password": "123456",
  "group_id": 1,
  "email": "1312342604@qq.com" // 注意，email为可选项
}

结果：

{
  "error_code": 0,
  "msg": "用户创建成功",
  "request": "POST  /user/register"
}

路由: admin/password/<int:id> put 方法 参数:

name 说明 类型 作用
new_password 新密码 string ****
confirm_password 确认密码 string ****

备注： 重置用户密码

发送:

{
  "new_password": "147258",
  "confirm_password": "147258"
}

结果：

{
  "error_code": 0,
  "msg": "密码修改成功",
  "request": "PUT  /cms/admin/password/2"
}

name 说明 类型 作用

name 说明 类型 作用

name 说明 类型 作用
