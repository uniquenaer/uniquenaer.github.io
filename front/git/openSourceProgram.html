#
开源项目踩坑记

辛苦几个月研发的微信小程序原生UI库终于要发布 v0.0.1-alpha.1 版本了，喜大普奔。然而一些平时想都没想过的问题迎面扑来，有些手忙脚乱，写个备忘，下次开源时可以更轻松上手。

# 1 多个远程仓库之间的管理

# 1.1 如何添加多个远程仓库

git remote add <name> <url>

name 表示你给远程仓库起的名字，url 表示这个库的地址

# 1.2 提交代码

git push <name> <branch>

name 表示1.1中给远程仓库起的名字，branch 表示要提交的某个分支
