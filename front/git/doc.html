#
玩转GIT 三剑客

# 开始

创建git仓库：git init

初始化用户信息（全局）：

git config --global user.name "John Doe"

git config --global user.email johndoe@example.com

初始化用户信息（仅当前仓库有效）：去除 --global 即可。或者 替换为 --local

# 暂存区 和 工作区

git add 将工作文件的修改提交到 暂存区

git commit 将暂存区的修改提交到 工作区

# 给文件重命名的简便方法

git mv 原文件 新文件名

# 查看git版本历史

git log --oneline

git log -n4 --oneline 查看几次的git提交

git log --all 查看所有分支的提交

git log --all --graph 图形化提交

# .git 解密

  • HEAD 当前在哪个分支
  • config 当前git仓库的配置信息
  • refs heads 和 tags
