# 原始（Primitive）类型

原始类型概念首次在这里接触，大概等同于基本类型吧

JS有6种基本类型：

  • string
  • number
  • boolean
  • undefined
  • null
  • symbol

首先原始类型存储的都是值，是没有函数可以调用的。Symbol是ES6新增的。

'1'.toString() 时，'1'被隐式转化为String对象。

Symbol - ECMAScript 6入门

对于 null 来说，虽然 typeof null 会输出 object，但是这只是 JS 存在的一个悠久 Bug。在 JS 的最初版本中使用的是 32 位系统，为了性能考虑使用低位存储变量的类型信息，000 开头代表是对象，然而 null 表示为全零，所以将它错误的判断为 object 。虽然现在的内部类型判断代码已经改变了，但是对于这个 Bug 却是一直流传下来。
